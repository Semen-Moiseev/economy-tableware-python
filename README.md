# economy-tableware-python

Экономика посуды

Небольшой проект на **Python + Tkinter + Matplotlib**, который помогает рассчитать, что выгоднее:

- мыть посуду самому;
- или использовать одноразовую.

Программа визуализирует зависимость затрат от количества использований и показывает **точку окупаемости**.

---

## Возможности

- Расчёт стоимости мытья посуды с учётом:
  - стоимости воды (горячей и холодной),
  - времени человека,
  - цены моющего средства.
- Сравнение с одноразовой посудой.
- Визуализация на графике:
  - динамика затрат;
  - точка, где становится выгоднее один из вариантов.

---

## Интерфейс

В приложении нужно ввести параметры:

| Поле                               | Описание                                    |
| ---------------------------------- | ------------------------------------------- |
| Количество посуды в наборе         | Сколько тарелок в наборе одноразовой посуды |
| Цена набора                        | Стоимость набора одноразовой посуды         |
| Стоимость холодной воды            | Цена за 1 м³ холодной воды                  |
| Стоимость горячей воды             | Цена за 1 м³ горячей воды                   |
| Время на мытьё тарелки             | Сколько секунд уходит на одну тарелку       |
| Стоимость часа вашего времени      | Сколько вы условно зарабатываете в час      |
| Объем бутылки моющего средства     | Сколько мл в бутылке                        |
| Стоимость бутылки моющего средства | Цена одной бутылки моющего                  |

После заполнения:

кнопка "Рассчитать" покажет, что выгоднее — мыть или одноразовая;

кнопка "Визуализировать" построит график зависимости затрат.
